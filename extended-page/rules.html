Upsilon GDPS Platformer List Rules

The Upsilon GDPS Platformer List is designed to maintain a fair and consistent environment for players submitting completions or trying to get new levels added to the list. The rules below outline the conditions under which a submission will be accepted or denied. These guidelines are updated regularly, so make sure to review them often.

Record Submission Rules (Cross-Platform)

These rules apply to ALL platforms unless otherwise specified.

  • A complete, uncut 0â€“100 run must be provided. If your main upload contains cuts, the raw footage MUST include the entire run with no interruptions.
  • After 2025, December 7th, Raw Footage is not required. But maybe if the level is hard, this is required. Also, the link should be shareable Google Drive link. We'll inspect the your completion if there's something suspicious. (Raw Footage will be kept private.)
  • You must show a reasonable number of previous attempts. For example, only dying once at 1% is not enough to prove legitimate progression. Everplay submissions are the only exception.
  • Endscreen stats must be fully visible (the entire box) for at least one frame.
  • If your mod menu supports a cheat indicator (such as Megahack v9, Eclipse or else.), it must be turned on and visible.
  • Clicks are required for all completions. Clicks must be consistent, audible, and match the onscreen actions.
  • Refer to this sheet for the full list of approved mods.
  • You must beat the correct version of the level. Verify the level ID on the Platformer List page before submitting. (LDM is allowed.)
  • No secret routes, no bug routes, and no exploits that noticeably change the intended difficulty.
  • No easy mode versions or modified copies of the level. Only the official published level is accepted.
  • Platformer completions are not allowed for Platformer List records.
  • Noclip accuracy runs are not accepted.
  • CBF (Click Between Frames) is allowed, but physics bypass is NOT. TPS bypass is allowed up to a maximum of 240 TPS.
  • FPS Bypass is allowed up to a maximum of 360 FPS.
  • When a level moves to the Legacy List, new records for that level will no longer be accepted.

    • Record Submission Rules (Mobile)

    Mobile players must also follow every rule listed above, with additional requirements to ensure legitimacy.

  • Visual taps are required. Your finger inputs must be clearly visible on the screen. If there's no visual taps, audible taps are required.

    • Level Rules

    These requirements must be met for a level to be considered for the UGDPS Platformer List.

  • The level must be at least 30 seconds long and rated Hard Demon or higher by the UGDPS moderators.
  • No crash triggers may be used. Any level containing crash triggers will be automatically denied.
  • If Random Triggers are used, the editor MUST clearly explain how the randomization works for transparency.
